## Представлять
Программа lab11cvdN3247 представляет собой утилиту командной строки, написанную на языке программирования C. 
Программа использует функцию opendir(), которая позволяет выполнять рекурсивный поиск файлов в заданном каталоге, соответствующих заданным условиям поиска.
## Системные Требования
- Операционная система Linux с версией ядра >= 5.0
- компилятор gcc и библиотека GNU C
- Valgrind для проверки ошибок памяти

## Скомпилируем и запустим программу
1. Убедитесь, что в вашей системе установлены компиляторы gcc и valgrind.
2. Откройте терминал и перейдите в папку, содержащую исходные файлы программы.
3. Скомпилируйте программу следующей командой:
valgrind ./lab11cvdN3247 <directory>

## Вариант
- Программа поддерживает следующие возможности:
- -h, --help: показать справку по опциям.
- -v, --version: Отобразить информацию о версии программы.
- Вы также можете установить переменную среды `LAB11DEBUG`, чтобы включить вывод отладочной информации.

----------------
lab11cvdN3247
----------------
Программа для выполнения рекурсивного поиска файлов,начиная с указанного каталога, в соответствии с условием из
Табл. 3 и вариантом из Табл. 4.

Порядок запуска:
$gcc -DLAB11DEBUG lab11cvdN3247 -o lab11cvdN3247.c

//
Символ препроцессора LAB11DEBUG можно определить во время компиляции, чтобы включить отладочную печать. Это может быть полезно для отладки и понимания поведения программы.
Если определен LAB11DEBUG, программа печатает дополнительную информацию во время выполнения.
//

$LAB11DEBUG=1 ./lab11cvdN3247 /home
//
Эта команда создаст исполняемый файл с именем lab11cvdN3247.
//

Применение
./lab11cvdN3247 [OPTION]... DIRECTORY SEARCH_TARGET

Пример запуска:
./lab11cvdN3247 /home/chu/Desktop/ 3
./lab11cvdN3247 /lib/ ab
